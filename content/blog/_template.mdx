# Your Blog Post Title

This is an example markdown blog post template for Finityo.

## Required Frontmatter Fields

- **slug**: URL-friendly identifier (e.g., "snowball-vs-avalanche")
- **title**: The main title of your post
- **description**: Brief excerpt for cards and SEO (150-160 characters)
- **date**: Publication date in YYYY-MM-DD format
- **category**: Post category (e.g., "Strategies", "Tips", "Guides")
- **image**: Path to header image (optional, defaults to /images/blog-default.png)
- **readTime**: Estimated reading time (e.g., "5 min read")

## Writing Your Post

Use standard Markdown syntax:

### Headings

Use `#` for h1, `##` for h2, etc.

### Lists

- Bullet point one
- Bullet point two
- Bullet point three

### Emphasis

You can use **bold** and *italic* text.

### Links

[Visit Finityo](https://finityo-debt.com)

### Code

Inline `code` or code blocks:

```javascript
const greeting = "Hello, world!";
console.log(greeting);
```

## How Slugs Work

The `slug` field in frontmatter determines the URL:
- Slug: `snowball-vs-avalanche`
- URL: `/blog/snowball-vs-avalanche`

Make sure slugs are unique across all blog posts!
