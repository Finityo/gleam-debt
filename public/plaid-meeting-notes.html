Plaid Integration Requirements

What We Need from Plaid:

  • Liabilities/Credit Data - Access to credit card and loan account information
    • Current balances
    • Minimum payment amounts
    • Payment due dates
    • Interest rates (APR)
  • Account Details - Basic account identification
    • Account names/types
    • Last 4 digits of account numbers
  • Read-Only Access - We only need to pull/read data, no payment initiation or transfers

Products Needed:

  • Liabilities (primary) - for credit cards and loans
  • Auth (basic) - for account verification

Use Case:

Debt payoff calculator that helps users create personalized debt repayment plans using the snowball/avalanche method

Data Refresh:

One-time pull when user connects + ability to refresh on-demand (not real-time monitoring)
